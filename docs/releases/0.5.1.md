## DeepSeek++ 0.5.1

0.5.1 是内置网络工具版本，重点让 DeepSeek 在需要实时信息或网页内容时能直接搜索、获取并继续生成。

### 主要变化

- 新增 `web_search` 和 `web_fetch` 内置工具，支持联网搜索与网页文本获取。
- 网络工具结果会回传到同一会话继续生成，适合实时信息查询、事实核验和网页总结。
- 侧边栏新增「工具」页，可开关网络工具、授权网页来源并运行搜索诊断。
- 搜索结果为空时会继续尝试可用搜索源，避免把不可解析页面误判为成功。
- 搜索规则只在 `web_search` 可用时注入，关闭工具后不会继续诱导模型调用。
- 修复工具续跑完成后的重复正文渲染，保留步骤记录并只展示一次最终回答。
