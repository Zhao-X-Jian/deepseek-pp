## DeepSeek++ 0.6.5

0.6.5 是中英文运行体验和输出反馈校准版本，让 DeepSeek++ 在中文或英文工作流里更一致，也让实时输出速度显示更贴近实际生成状态。

### 主要变化

- 语言选择继续覆盖侧边栏、右键菜单、工具结果、内置 Skill 和模型续跑提示，中文或英文体验更一致。
- 用户创建的记忆、预设、自定义 Skill、自动化任务和同步数据继续保留原文，不会因为切换界面语言被翻译或改写。
- `tok/s` 估算按更贴近 DeepSeek 输出特征的方式校准，并从首个流式内容后开始计速，减少排队和预填充延迟造成的偏差。
- README 和商店材料同步 Chrome Web Store 可用状态，并继续保留 Shell MCP 与本机工具安装说明。
- 发布门禁加入多语言覆盖校验，继续覆盖编译、单测、MCP、自动化、提示词冻结、多浏览器打包、workflow 和资产校验。

感谢本版本贡献者：[@mekos2772](https://github.com/mekos2772) 修复 Token 输出速度估算偏差。
